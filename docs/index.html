📱 iPhone / iPad

  3. Cole a URL → Assinar
🤖 Android

⚠️ O app Google Calendar e o Samsung Calendar não permitem assinar calendários por URL diretamente no celular.

Opção 1 — Google Calendar na web (recomendado)

  1. Abra calendar.google.com no computador ou ative "Ver como site para desktop" no navegador do celular
  2. No menu lateral esquerdo, clique + ao lado de "Outros calendários"
  3. Selecione "De URL"
  4. Cole a URL → Adicionar calendário

⚠️ A versão mobile do site não mostra essa opção — use um computador ou ative o modo desktop no navegador

✓ Após adicionar, sincroniza automaticamente com o app do celular (Google Calendar e Samsung Calendar)

Opção 2 — App ICSx5 (sem conta Google)

  1. Instale ICSx5 na Play Store
  2. Abra o app e toque +
  3. Cole a URL → configurar intervalo de atualização

✓ Funciona em qualquer Android, inclusive sem Google

💻 Outlook
  1. Calendário → Adicionar calendário
  2. Assinar da web
  3. Cole a URL → Importar
